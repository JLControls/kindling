# Kindling Documentation

Welcome to the Kindling documentation. Kindling is a standalone desktop application for advanced [Ignition](https://inductiveautomation.com/) users, providing various tools to read and access Ignition's data export formats.

## Features

- **Thread Viewer** - Parse and analyze Ignition thread dumps
- **IDB Viewer** - Explore SQLite databases with special Ignition handling
- **Log Viewer** - Aggregate and analyze wrapper.log files
- **Archive Explorer** - Browse .gwbk, .modl, and other ZIP archives
- **Git Branch Comparison** - Compare Ignition configurations across branches
- **CLI Mode** - Command-line interface for automation and LLM integration

## Quick Start

1. Download the installer for your OS from the [Downloads page](https://inductiveautomation.github.io/kindling/download.html)
2. Run the Kindling application
3. Open a supported file by:
   - Drag and drop onto the application window
   - Click the `+` icon in the tab strip
   - Select a tool from the menu bar
